
#INSTALL AND RUN APP 

### setup golang

  install golang

  export gopath, example

  `export GOPATH=~/go`

  `go get github.com/hypebeast/go-osc/osc`

  `go run main.go`

  `go build main.go`

### Run application

tab1: `npm run-script watch`

tab2: `npm start`

### Build static assets

`webpack`
